# HR-Analytics-Project-Workforce-Intelligence-Business-Insights
This project analyzes 8,950 employee records to uncover critical workforce insights using SQL. The goal is to transform raw HR data into actionable insights that help organizations improve hiring strategy, employee retention, and performance management.

## Project Overview
This project analyzes 8,950 employee records to uncover critical workforce insights using SQL.
The goal is to transform raw HR data into actionable insights that help organizations improve hiring strategy, employee retention, and performance management.

Dataset Summary
•	👥 Total Employees: 8,950
•	🏢 Departments: Multiple (HR, Sales, IT, Marketing, Operations, etc.)
•	💼 Job Titles: 28 unique roles
•	🟢 Active Employees: 7,984
•	🔴 Employees Left: 966

Workforce Overview
Out of 8,950 employees:
•	89.2% employees are still active
•	Workforce is distributed across multiple departments and job roles
